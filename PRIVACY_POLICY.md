# Clockboard Privacy Policy

Clockboard does not collect, sell, transmit, or share personal data.

Clockboard stores only local extension settings: clock preferences, Clockboard item order and visibility, countdown names, and countdown local date/time targets. Settings use browser extension storage, with sync storage when available and local storage as a fallback.

Clockboard makes no external network requests and includes no analytics, telemetry, remote scripts, remote images, host permissions, or tracking code.

Clockboard requests only the `storage` permission so it can save settings and countdowns.
