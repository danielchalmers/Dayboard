# Clockboard Privacy

Clockboard does not collect, sell, transmit, or share personal data.

## Data Stored

Clockboard stores only extension settings needed for the user experience:

- Clock display preferences.
- Countdown names.
- Countdown local date/time targets.
- Active countdown selection.

Settings are stored with the browser extension storage API. Clockboard uses sync storage when available and local storage as a fallback.

## Network Use

Clockboard does not make external network requests. It has no analytics, telemetry, remote scripts, remote images, host permissions, or tracking code.

## Permissions

Clockboard requests only the `storage` permission. This permission is required to save clock preferences and countdowns.
